## üîß How It Works

### 1. Web Scraping (FREE)
Monitors all major platforms without expensive APIs:
- **Twitter**: Nitter mirrors (free vs $5,000-$42,000/month Twitter API)
- **Reddit**: PRAW library (60 requests/min free)
- **Instagram**: Instaloader (no API needed)
- **News**: Google News RSS + NewsAPI.org (100/day free)
- **TikTok**: TikTok-Api library (free)

### 2. AI Detection (FREE)
Uses open-source HuggingFace models:
- **Fake News**: BERT classifier (free vs $2,000-$40,000/month for GPT-4)
- **Sentiment**: RoBERTa sentiment analyzer
- **Deepfakes**: ELA + EXIF analysis
- **Impersonation**: Zero-shot classification

### 3. Real-Time Alerts
- Scans every 15 minutes during business hours
- Comprehensive daily report at 2 AM
- Email alerts via SendGrid (100/day free)
- Threat severity: Low/Medium/High/Critical

### 4. Evidence-Based Reports
- Screenshots of threatening content
- Source URLs and timestamps
- AI confidence scores
- Recommended actions
- Daily PDF summaries
